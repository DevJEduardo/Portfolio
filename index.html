Cajui Notification

O Cajui Notification é uma solução abrangente que consiste em um aplicativo móvel e um site para simplificar o acesso a informação. O aplicativo permite que os usuários recebam, visualizem e interajam facilmente com as notificações em tempo real, enquanto o site oferece uma plataforma centralizada para cadastrar e gerenciar notificações e todos os usuarios com acesso ao site.

Banco

O Banco é uma sistema que foi criado como projeto do IFNMG afim de simular um backend de um banco real com cadastro de clientes, contas, bancos e agencias. A principio foi proposto somente a validação de CPF, porém enquanto o desenvolvia introduzi validações de cadastros existentes para outras telas serem acessadas, isso sem a ultilização de um banco de dados, sendo todo os dados armazenados em "listas no java" ou "Arraylist".
